# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

> 💡 These are just the design sizes. Ensure content is responsive and meets WCAG requirements by testing the full range of screen sizes from 320px to large screens.

## Colors

### Primary

- Light red: hsl(0, 100%, 67%), oklch(0.6843 0.2045 24.3)
- Orangey yellow: hsl(39, 100%, 56%), oklch(0.8139 0.1654 75.39)
- Green teal: hsl(166, 100%, 37%), oklch(0.7085 0.1426 168.5)
- Cobalt blue: hsl(234, 85%, 45%), oklch(0.4208 0.2542 265.77)

## Gradients

- Light slate blue (background): hsl(252, 100%, 67%),oklch(0.5884 0.2364 285.76)
- Light royal blue (background): hsl(241, 81%, 54%), oklch(0.4611 0.2661 270.02)

- Violet blue (circle): hsla(256, 72%, 46%, 1), oklch(0.4388 0.2322 283.49)
- Persian blue (circle): hsla(241, 72%, 46%, 0), oklch(0.4102 0.2416 269.47 / 0%)



### Neutral

- White: hsl(0, 0%, 100%), oklch(1 0 0)
- Pale blue: hsl(221, 100%, 96%), oklch(0.9575 0.02 265.9)
- Light lavender: hsl(241, 100%, 89%), oklch(0.8491 0.0778 285.38)
- Dark gray blue: hsl(224, 30%, 27%), oklch(0.3576 0.0549 268.3)

### Notes

Use transparency to get the color variations necessary to match the design. Hint: look into using `hsla()`.
